# Carpeta de Imágenes

Coloca aquí tus imágenes decorativas:

## Estructura recomendada:

```
images/
├── hero-bg.jpg        # Imagen de fondo para el hero section
├── avatar.png         # Tu foto/avatar circular
└── oc/               # Carpeta para imágenes de tu OC
    ├── oc-1.png
    ├── oc-2.png
    └── ...
```

## Formatos recomendados:
- **Hero background**: JPG/WebP (1920x1080px o mayor)
- **Avatar**: PNG con fondo transparente (500x500px)
- **OC images**: PNG/JPG (depende del uso)

## Optimización:
Next.js optimiza automáticamente las imágenes cuando usas el componente `<Image>`.
